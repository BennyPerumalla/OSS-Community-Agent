# Getting Started with Our Project

Welcome! To get started, you'll need to clone the repository from GitHub.

## Requirements
- Python 3.11+
- `uv` (recommended)
- A Reddit API account

## Installation
1. Clone the repo: `git clone <repo_url>`
2. Navigate to the directory: `cd oss-community-agent`
3. Install dependencies: `uv pip install -r infra/requirements.txt`
